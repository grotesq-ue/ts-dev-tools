# @grotesq-ue/ts-dev-tools

A simple CLI application for scaffolding grotesq-ue preferred TypeScript Development Tools.

## Tools

- Linter: [oxlint](https://oxc.rs/docs/guide/usage/linter.html)
- Formatter: [dprint](https://dprint.dev/)
- Git Hook Manager: [lefthook](https://lefthook.dev/)

## Technology Stack

- Language: [TypeScript](https://www.typescriptlang.org/)
- Environment: [Bun](https://bun.com/)
- Build Tool: [Bunup](https://bunup.dev/)
- Dependencies:
  - [package-manager-detector](https://github.com/antfu-collective/package-manager-detector)
  - [js-yaml](https://github.com/nodeca/js-yaml)
